# Data Sources

Below are examples of public or community threat data sources. Always review terms of use and rate limits.

## Examples

- AbuseIPDB
- AlienVault OTX (REST API)
- AlienVault OTX (TAXII/STIX)
- AlienVault OTX OTXv2 SDK
- CIRCL MISP feeds
- MalwareBazaar
- Shadowserver public reports
- Shodan (paid)

## Notes

- Prefer sources that allow redistribution if you plan to publish data.
- Keep samples small and anonymized when committing to the repo.
